# AI 발표 대본 생성기

## 핵심 기능
발표 주제와 필수 내용을 입력하고 발표 스타일과 시간을 선택하는 등 발표에 필요한 정보를 입력하면 AI가 발표대본을 자동으로 생성한다.

**연습 모드 기능**을 통해 실전처럼 발표를 진행할 수 있다. (프롬프터, 음성 녹음 피드백, 타이머)

## 실행 방법
- Google AI Studio에서 무료 Gemini API Key를 받아 API 설정에 입력한다.

- 발표 제목, 세부 주제 등 대본의 기본 정보를 입력한다.

- 서론, 본론, 결론에 필수적으로 포함되어야 할 내용을 입력한다.

- 1분, 3분, 5분, 10분, 15분 중 해당되는 발표 시간을 선택한다.

- 차분한 스타일, 친근한 스타일, 공식적인 스타일, 설득형 스타일 중 원하는 말하기 스타일을 선택한다.

- AI를 통해 생성한 대본은 전체보기와 구조별 보기 중 원하는 형식으로 볼 수 있다.

- 대본 편집 기능을 통해 대본을 수정할 수 있으며, 텍스트 다운로드 및 대본 복사하기 기능을 통해 대본 내용을 복사할 수 있다.

- AI 다시 생성 기능을 통해 대본 전체를 재생성할 수 있다.

- 발표 연습하기(프롬프터)에서 발표 시작 버튼을 누르면 타이머가 작동한다. 글자 크기, 스크롤 속도를 조절하여 원하는 프롬프터를 설정할 수 있다. 내 목소리 녹음하기 버튼을 누르면 나의 발표를 녹음하고 객관적으로 평가할 수 있다.

## 기대 효과
쉽고 빠르게 대본을 만들어 발표 준비 시간을 줄일 수 있고, 발표 경험이 부족한 사용자들도 연습 모드 기능을 통해 발표 실력을 늘릴 수 있다.
